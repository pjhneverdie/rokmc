# The Document Object
document는 html 트리를 js Object로 가지고 있는 거, 실제 Object 요소 수정하는 것처럼 js로 html 부분부분을 수정할 수 있음.

js의 document를 사용해서 html을 조작할 수 있는 방법은 더러 있음. ```querySelector``` 가 유용할 듯

# event
js로 웹페이지 상에서 발생하는 event를 listen할 수 있음.
https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement (JS로 들을 수 있는 이벤트 모음집)

```js
const title = document.querySelector("div.hello:first-child h1");

function handleTitleClick (){
    console.log("title was clicked");
}

title.addEventListener("click", handleTitleClick);
```

# window
웹 브라우저가 window 객체를 만들어서 js 전역 변수로 등록해줌.
그래서 js로 웹브라우저의 event를 읽을 수 있음.

https://developer.mozilla.org/ko/docs/Web/API/Window (JS로 들을 수 있는 이벤트 모음집)

```js
function handleWindowResize(){
    document.body.style.backgroundColor = "tomato"
}

window.addEventListener("resize", handleWindowResize);
```
